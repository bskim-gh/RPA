🔧 B. 솔루션 분석 업무 - 글로벌 물류 솔루션 분석

🎯 글로벌 물류 솔루션 분석 성과

업무 영역 담당 업무 관리 방식 준비 성과
솔루션 분석 Shippo, ShipStation, EasyShip 3개 솔루션 종합 분석 API 연동성, 국가별 서비스 범위, 비용 구조 비교 K2J 물류에 최적화된 솔루션 선정 및 도입 로드맵 수립
보고자료 제작 글로벌 물류 프로세스에 대한 Shippo 기능 분석 및 특송사와의 R&R 점검, 솔루션 비교표 제작 Figma 프로토타입, 시각적 다이어그램, 매뉴얼 작성 실무진 활용 가능한 시각적 자료 및 가이드북 완성
사내 공유 분석 결과 및 솔루션 도입 방안 사내 공유 팀 미팅 프레젠테이션, 보고서 배포, Q&A 세션 팀 내 지식 공유 및 솔루션 도입 의사결정 지원

📊 분석 개요

글로벌 물류 솔루션 시장의 주요 플레이어인 Shippo, ShipStation, EasyShip을 대상으로 서비스 범위, 기능, 비용 구조를 종합적으로 분석하여 최적의 솔루션 도입 방안을 수립했습니다.

S
Shippo

글로벌 물류 API 플랫폼

✅ 장점
  • 글로벌 서비스 (국가 제한 없음)
  • 30일 무료체험 제공
  • 5개 설치솔루션 지원
  • API 연동 용이
⚠️ 제약사항
  • 택배 서비스 제한 (2업체)
  • 계정 제한 설정 불가
  • 개인계정만 지원

💡 추천 용도: 글로벌 서비스가 필요한 중소기업, API 중심의 자동화 시스템 구축

ShipStation

전문 물류 관리 솔루션

✅ 장점
  • 마스터계정 + 에이전시 지원
  • 계정 제한 설정 가능
  • 4업체 택배 서비스
  • 가입 후 30일 무료
⚠️ 제약사항
  • 6개국 서비스 (한국 제외)
  • 설치솔루션 2개만 지원
  • 복잡한 계정 구조

💡 추천 용도: 대기업, 복잡한 물류 관리가 필요한 기업, 계정 관리가 중요한 경우

  • 간편한 사용법
⚠️ 제약사항
  • 택배 서비스 불가
  • 설치솔루션 2개만 지원
  • 계정 제한 설정 불가

💡 추천 용도: 한국 시장 진출, 간단한 물류 관리가 필요한 소규모 기업

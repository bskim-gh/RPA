🔧 Hello Tool 템플릿

Tool 구현 템플릿

@mcp.tool()
async def hello(name: str, term: Optional[str] = None) -> Dict[str, Any]:
    """MCP Tool 템플릿 - 인사말 생성 도구"""
    
    greeting = {
        "morning": "좋은 아침입니다",
        "afternoon": "좋은 오후입니다",
        "evening": "좋은 저녁입니다"
    }.get(term, "안녕하세요")
    
    return {
        "status": "success",
        "message": f"{greeting}, {name}님!",
        "detail": {"timestamp": datetime.now().isoformat()}
    }

Tool 명세

속성 설명
Tool Name hello
Full Identifier mcp-template--hello
Parameters • name (str, required): 인사할 대상 이름
• term (str, optional): 시간대 정보
Return Type Dict[str, Any] - status, message, detail을 포함한 응답
Error Handling 예외 발생 시 error status와 함께 에러 메시지 반환

🚀 FastMCP 서버 구현

메인 서버 구현 (main.py)

from fastmcp import FastMCP
from sse_starlette.sse import EventSourceResponse
from starlette.applications import Starlette
from starlette.routing import Route, Mount

# FastMCP 서버 초기화
mcp = FastMCP("mcp-template")

# SSE 연결 핸들러
async def handle_sse(request):
    async def event_generator():
        while True:
            yield {"event": "message", "data": json.dumps({"type": "heartbeat"})}
            await asyncio.sleep(30)
    return EventSourceResponse(event_generator())

# Mount 구조로 담당자별 인스턴스 분리
app = Starlette(
    routes=[
        Route("/sse", endpoint=handle_sse),
        Mount("/messages/", app=sse.handle_post_message),
        
        # 담당자별 독립 MCP 인스턴스 Mount
        Mount("/mcp-user1/", app=user1_mcp_app),  # 담당자1 MCP
        Mount("/mcp-user2/", app=user2_mcp_app),  # 담당자2 MCP
        Mount("/mcp-team-a/", app=team_a_mcp_app), # 팀A MCP
        Mount("/mcp-team-b/", app=team_b_mcp_app), # 팀B MCP
    ],
)

if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)

🔒 보안 설정

컨테이너 보안
  • ✅ 비루트 사용자(UID 1000) 실행
  • ✅ SetUID/SetGID 권한 제거
  • ✅ 최소 권한 원칙 적용
  • ✅ 불필요한 시스템 패키지 제거
네트워크 보안
  • ✅ 단일 포트(8000)만 노출
  • ✅ 헬스체크 엔드포인트 구현
  • ✅ SSL/TLS 지원 (리버스 프록시)
  • ✅ Rate limiting 지원
